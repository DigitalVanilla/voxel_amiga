W - forward
S - backward

A - Strife left
D - Strife right

Arrow up - raise in height
Arrow down - lower in height

Q - more FOV render distance
E - less FOV render distance


Delta Z resolution is how many lines taken in account
from the camera to the FOV distance value.
Higher values means less lines rendered, lower values means more lines rendered:

Z - Add Delta Z value resolution (more is equal to less details)
X - Remove Delta Z value resolution (less is equal to more details)


Makes things more or less tall:

C - Scale the height value up
V - Scale the height value down


Horizontal divisions resolution (to speed up things):

L - Cycle between 1 and 4


Extra:

N - Night mode
K - Debug output
